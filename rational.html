Rational Numbers

Description

Function Reference

rational(numerator: number, denominator: number): RationalNumber

Used to create a RationalNumber. Returns a new RationalNumber with the supplied numerator and denominator.

addRational(a: RationalNumber, b: RationalNumber): RationalNumber

Takes in two RationalNumbers and returns a new RationalNumber which is equal to them added.

subtractRational(a: RationalNumber, b: RationalNumber): RationalNumber

Takes in two RationalNumbers and returns a new RationalNumber which is equal to the second subtracted from the first.

divideRational(a: RationalNumber, b: RationalNumber): RationalNumber

Takes in two RationalNumbers and returns a new RationalNumber which is equal to the first divided by the second.

multiplyRational(a: RationalNumber, b: RationalNumber): RationalNumber

Takes in two RationalNumbers and returns a new RationalNumber which is equal to them multiplied together.

isSmallerThan(a: RationalNumber, b: RationalNumber): RationalNumber

Compares the two RationalNumbers supplied as parameters and if the first is smaller it returns true, otherwise it will return false.

isBiggerThan(a: RationalNumber, b: RationalNumber): boolean

Compares the two RationalNumbers supplied as parameters and if the first is bigger it returns true, otherwise it will return false.

isEqualTo(a: RationalNumber, b: RationalNumber): boolean

Compares the two RationalNumbers supplied as parameters and if they are equal it returns true, otherwise it will return false.

fractionString(x: RationalNumber): string

Returns a fractional representation of the RationalNumber in the format: "[numerator]/[denominator]".

decimalString(x: RationalNumber, decimalPlaces: number): string

Returns a decimal representation of the RationalNumber in the format: "x.xxxxxx" (to decimalPlaces decimal points).
