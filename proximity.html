Proximity

Description

The proximity package helps users to estimate the distance between two microbits. It works by measuring the signal strength between two microbits and then using that value to estimate how far away the microbits are. This is achieved by comparing the signal stregth to the distance we would expect based using real world data that has been collected.
Avaliable here:
https://makecode.microbit.org/_TgWaRT5KV5jM

Function Reference

Proximity.BroadcastProximity()

This function broadcasts a message to all other connected microbits in the group. This can be used to then find the signal strength.

Proximity.onReceived( ({ distance, strength })

This function is an event that is activated whenever a message is received. It provides the distance and signal strength for the user to use. This can be used for example to determine how far away a microbit that had just sent a message is.

Proximity.StrengthToDistance(int SignalStregth)

Uses the signal strength of the last received message to calculate and return an integer value that is the estimated distance that that message travelled.
